LQ model, α/β = 3 Gy. Near-max scope = dose-limit entries with Volume ≤ 0.1 cc. CV scope = complementary-volume (cold-volume) entries for parallel organs (Liver, Kidney_Cortex, Lungs), interpreted as "minimum volume spared from ≥ dose." Filters: pre-converted EQD2 or BED, α/β = 2.5, and re-irradiation intent excluded. Conventional-fractionation reference doses are treated as EQD2-equivalent, since at ~1.8-2 Gy/fx the physical dose equals EQD2 by definition. Near-max reference = highest Gy among records with QUANTEC tag and Fractions = Conventional. CV reference = conventional-fractionation CV value for that OAR (Liver 36 Gy ≥ 700 cc, Kidney_Cortex 27 Gy ≥ 200 cc, Lungs 18 Gy ≥ 950 cc).
